Commands Reference Sheet:

Terminal Commands:

1. pwd: Present Working Directory:
prints the current working director

2. ls: List:
lists the current files and Directory.

3. cd: Change Directory:
Changes the current directory to what is spesicied by the user.

4. mkdir: Make New Directory:
creates a new, empty directory.

5. mv: move command:
to move files and directories from one directory to another

6. touch:
used to create new, empty files

7. vi:
text editor that can be used to edit files through the terminal/Comand Line.

Git Commands:

1. git init:
Creates a new git repo

2. git add (filename):
adds a change in the working directory to the. If you include a ".", then it will include all the files.

3. git commit -m (message describing commit)
commits your changes with a message that discribes the changes that you are commiting

4. git checkout (-b) (branch) (-b makes new branch and switches to it)

5. git push (remote) (branch)

Docker Commands:

1. docker compose up
Starts containers and creates containers, networks, volumes, and images created

2. docker compose down
Stops containers and removes containers, networks, volumes, and images created by up

3. Docker ps
lists the running containers

4. docker kill (container id):
Kills one or more containers.
